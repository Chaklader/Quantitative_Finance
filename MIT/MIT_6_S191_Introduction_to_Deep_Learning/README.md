# MIT 6.S191: Introduction to Deep Learning (2025 Edition)

#### Course Overview

| Aspect            | Details                                                      |
| ----------------- | ------------------------------------------------------------ |
| Program Type      | Intensive bootcamp on deep learning fundamentals             |
